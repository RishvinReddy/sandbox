## 🔬 What Is This Repo?
```
sandbox/
├── It's messy          → intentional
├── It's experimental   → always
├── It might be broken  → probably
└── It's mine           → definitely
```

This is **Rishvin's personal sandbox** — a dedicated space where I test, break, rebuild, and explore before anything makes it to a real project. Nothing here is meant to be polished. Everything here is meant to be *tried*.

## 🚦 Reading the Commits

My commit messages in this repo follow a loose convention:

| Prefix | Meaning |
|--------|---------|
| `test:` | Something I'm trying out |
| `wip:` | Work in progress, not done |
| `fix:` | Fixed something I broke |
| `nope:` | Reverting an idea that didn't work |
| `ok this works:` | Rare moment of victory |
